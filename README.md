# IkaGX

> **Ikariam Intelligence.**

IkaGX é uma extensão para Google Chrome desenvolvida para ampliar a experiência no Ikariam através de ferramentas inteligentes, informações adicionais e recursos que tornam o gerenciamento do império mais prático e eficiente.

O projeto tem como objetivo oferecer uma experiência integrada ao jogo, mantendo uma interface simples, rápida e intuitiva.

---

## ✨ Recursos

- Histórico de IPs
- Consulta de contas por Lobby
- Identificação automática do servidor
- Interface integrada ao Ikariam
- Armazenamento local dos dados
- Janelas modais para consulta das informações

---

## 📦 Instalação

1. Clone este repositório.

```bash
git clone https://github.com/SEU_USUARIO/ikagx.git
```

2. Abra o Google Chrome.

3. Acesse:

```text
chrome://extensions
```

4. Ative o **Modo do Desenvolvedor**.

5. Clique em **Carregar sem compactação**.

6. Selecione a pasta do projeto.
